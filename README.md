# どこでもバッファ

レイヤー間で映像を送受信できるカスタムオブジェクトを追加します

beta28aにて動作確認済

# 導入手順

1. [リリースページ](../../releases/latest/)から `@どこでもバッファ.obj2` をダウンロード
2. AviUtl2の画面にD&D
3. `このプラグイン・スクリプトを信頼して使用する` をクリック
4. `カスタムオブジェクト > どこでもバッファ` に追加されます

# 使い方

### `入力` および `出力` オブジェクト
- `入力` オブジェクトが置かれたレイヤーまでの映像データを、 `出力` オブジェクトに送信することができます
- `出力` より上のレイヤーに `入力` を置いた場合でも動作します
  - ただし、オブジェクトの出力映像は1フレーム遅れた状態になります
- それぞれのオブジェクトには `ID` が指定されており、同じ `ID` が指定されたオブジェクト同士で送受信が行われます
  - `ID` が `0` の `入力` オブジェクトから送信された映像は、同じく `ID` が `0` のオブジェクトでのみ受信できます
- `ID` には任意の文字列を設定できます

### `遅延` オブジェクト

- 指定したフレーム数だけ映像を遅延させます
  - 1オブジェクトあたり `フレーム数 x シーン解像度` に応じたメモリを必要とします
